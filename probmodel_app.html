Categorizzazione del rapporto loan-to-value

Classifichiamo il rapporto loan-to-value (lvr) in categorie basse, medie e alte per analisi future.

Hmda <- Hmda |> 
  mutate(lvra = case_when(
    lvr < 0.8 ~ "low",
    lvr > 0.95 ~ "high",
    TRUE ~ "medium"
  ))

Creiamo la variable dummy denied che prende il valore 1 quando deny=="yes" and il valore 0 quando deny=="no".

Hmda <- Hmda |>
  mutate(denied = case_when(
    deny == "yes" ~ 1,
    deny == "no" ~ 0
  ))

Il coefficient sulla variable blackyes (la dummy automaticamente creata da R che prende il valore 1 se black=="yes") √® 0.085. L‚Äôinterpretazione di questo coefficiente √® che una persona di coloro a parit√† di altre caratteristiche ha una probabilit√† di avere il mutuo concesso pi√π bassa di 0.085.

Per avere un‚Äôidea di quanto sia grande questa differenza, possiamo calcolare la variazione relative della probabilit√†: \[ \frac{\Pr(black=1|X) - \Pr(black=0|X)}{\Pr(black=0)} = \frac{0.085}{0.093} \approx 0.91 (91\%) \] In altre parole, se rapportata alla probabilit√† (non condizionata) che hanno le persone non di colore di ottenere il mutuo (9.3%), le persone di colore hanno una probabilita‚Äô piu bassa di ottenere il mutuo di circa il 90%.

Modello Logit e Probit

logit_HMDA <-
  feglm(
    denied ~ black + dir + hir + lvra + ccs + mcs + pbcr +
      dmi + self + uria + condominium,
    data = Hmda,
    family = binomial("logit")
  )
probit_HMDA <-
  feglm(
    denied ~ black + dir + hir + lvra + ccs + mcs + pbcr +
      dmi + self + uria + condominium,
    data = Hmda,
    family = binomial("probit")
  )

Confrontiamo i tre diversi modelli:

Mentre il coefficiente del modello lineare per la probabilit√† si presta immediatamente ad una interpretazione, i coefficienti del modello logit e probit ci danno esclusiavamente delle informazioni riguardo al segno della relazione fra la probabilit√† e le rispettive variabili. Per ottenere un‚Äôinterpretazione bisogna procedere calcolando le predizioni.

Predizioni

Il modo pi√π semplice per ottenere una stima della differenza nella probabilita‚Äô che il mutuo venga accettto per le persone di colore e quelle non di colore e‚Äô procedere a calcolare le predizioni a dei valori prespecificati delle \(X\):

Per il modello probit: \[ \begin{aligned} \underset{\text{probabilit√† diniego mutuo per black}}{\underbrace{\Phi\left(\hat{\beta}_{0}+\hat{\beta}_{1}+\hat{\beta}_{2}\bar{dir}+\hat{\beta}_{3}\bar{hir}+\hat{\beta}_{6}\bar{ccs}+\hat{\beta}_{7}\bar{mcs}+\hat{\beta}_{11}\bar{uria}\right)}}\\-\underset{\text{probabilit√† diniego mutuo per non-black}}{\underbrace{\Phi\left(\hat{\beta}_{0}+\hat{\beta}_{2}\bar{dir}+\hat{\beta}_{3}\bar{hir}+\hat{\beta}_{6}\bar{ccs}+\hat{\beta}_{7}\bar{mcs}+\hat{\beta}_{11}\bar{uria}\right)}} \end{aligned} \] e per il modello logit \[ \begin{aligned} \underset{\text{probabilit√† diniego mutuo per black}}{\underbrace{F\left(\hat{\beta}_{0}+\hat{\beta}_{1}+\hat{\beta}_{2}\bar{dir}+\hat{\beta}_{3}\bar{hir}+\hat{\beta}_{6}\bar{ccs}+\hat{\beta}_{7}\bar{mcs}+\hat{\beta}_{11}\bar{uria}\right)}}\\-\underset{\text{probabilit√† diniego mutuo per non-black}}{\underbrace{F\left(\hat{\beta}_{0}+\hat{\beta}_{2}\bar{dir}+\hat{\beta}_{3}\bar{hir}+\hat{\beta}_{6}\bar{ccs}+\hat{\beta}_{7}\bar{mcs}+\hat{\beta}_{11}\bar{uria}\right)}}, \end{aligned} \] dove \(F(\cdot)\) √® la funzione di ripartizaione della distribuzione logistica.

Per ottenere le predizioni √® necessario creare un data.frame contenente i valori per le \(X\):

new <- data.frame(
  "dir" = mean(Hmda$dir),
  "hir" = mean(Hmda$hir),
  "lvra" = "low",
  "ccs" = mean(Hmda$ccs),
  "mcs" = mean(Hmda$mcs),
  "pbcr" = "no",
  "dmi" = "no",
  "self" = "no",
  "black" = c("no", "yes"),
  "uria" = mean(Hmda$uria),
  "condominium" = 0
)

Possiamo ottenere le predizioni per i due modelli usando la funzione predict:

logit_pred <- predict(logit_HMDA, newdata = new)
probit_pred <- predict(probit_HMDA, newdata = new)

logit_pred e probit_pred sono due vettori che hanno come prime elemento la stima della probabilit√† per black and come secondo la stima della probabilit√† per non-black (condizionatamente ai valori specificati in new per le altre variabili). La differenza di questi due valori √® la stima della differenza della probabilita‚Äô.

delta_logit = diff(logit_pred)
